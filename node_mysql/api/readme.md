### Criação do projeto

* package.json
```
npm init -y
```

### Dependências

* 1. Express
```
npm i express
```

* 2. Sequelize
```
npm i --save sequelize
```

* 3. Mysql2
```
npm i --save mysql2
```

* 4. Bcrypt
```
npm i bcrypt
```

* 5. JWT
```
npm i --save jsonwebtoken
```

* 6. DotEnv gerenciar variaveis de ambiente
```
npm i dotenv
```

* 7. CORS Liberar acesso externo
```
npm i cors
```
